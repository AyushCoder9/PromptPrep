# PrepAI (Idea Document)

## Problem

Students drown in unstructured PDFs and struggle to test themselves.

## Solution

An AI backend that converts static notes into interactive quizzes and flashcards.

## Key Features

- Document Ingestion (PDF/Text)
- Vector Search
- Quiz Generation Engine
- Flashcard API

## Tech Stack

- **Languages:** Python (FastAPI), React (Frontend)
- **Frameworks:** LangChain
- **AI/ML:** OpenAI/Gemini API, FAISS/ChromaDB (Vector Database)
